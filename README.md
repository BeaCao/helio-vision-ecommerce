## 🇺🇸 English

![HELIO Vision Pro](https://raw.githubusercontent.com/BeaCao/helio-vision-ecommerce/main/src/assets/hero.png)

An ultra-luxury e-commerce experience inspired by spatial computing. This project recreates a high-end online store with a minimalist design, fluid animations, and an exceptional user experience.

### ✨ Key Features

- 🎨 **Interactive Configurator**: Customize your HELIO Vision Pro with real-time changes in color, storage, and accessories.
- 🌍 **Multi-language Support (i18n)**: Fully translated into 5 languages: **English, Spanish, French, German, and Japanese**.
- 📱 **Pro Responsive Design**: Meticulously optimized for mobile, tablet, and desktop with a "native app" style interface.
- 🛒 **Shopping Ecosystem**:
  - Shopping cart with quantity management.
  - Persistent Wishlist.
  - "Social Proof" notifications (real-time recent sales).
- 🎭 **Cinematic Animations**: **GSAP** implementation for smooth scrolling and premium reveal effects.
- 💬 **Live Support**: Integrated interactive chat widget.

### 🛠️ Tech Stack

- **Framework**: [Vue.js 3](https://vuejs.org/) (Composition API)
- **Tooling**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/)

### 🚀 Installation

```bash
git clone https://github.com/BeaCao/helio-vision-ecommerce.git
npm install
npm run dev
```
