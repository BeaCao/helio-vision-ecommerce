# 🥽 HELIO Vision Pro — Premium E-commerce Storefront

![HELIO Vision Pro](https://raw.githubusercontent.com/BeaCao/helio-vision-ecommerce/main/src/assets/hero.png)

Una experiencia de comercio electrónico de ultra-lujo inspirada en la computación espacial. Este proyecto recrea una tienda online de gama alta con un diseño minimalista, animaciones fluidas y una experiencia de usuario excepcional.

## ✨ Características Principales

- 🎨 **Configurador Interactivo**: Personaliza tu HELIO Vision Pro con cambios de color, almacenamiento y accesorios en tiempo real.
- 🌍 **Soporte Multi-idioma (i18n)**: Traducido completamente a 5 idiomas: **Español, Inglés, Francés, Alemán y Japonés**.
- 📱 **Diseño Responsive Pro**: Optimizado meticulosamente para móviles, tablets y escritorio con una interfaz estilo "App nativa".
- 🛒 **Ecosistema de Compra**:
  - Carrito de compra con gestión de cantidades.
  - Lista de deseos (Wishlist) persistente.
  - Notificaciones de "Prueba Social" (ventas recientes en tiempo real).
- 🎭 **Animaciones de Cine**: Implementación de **GSAP** para scroll suave y efectos de revelación premium.
- 💬 **Soporte en Vivo**: Widget de chat interactivo integrado.

## 🛠️ Tecnologías Utilizadas

- **Framework**: [Vue.js 3](https://vuejs.org/) (Composition API)
- **Tooling**: [Vite](https://vitejs.dev/)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones**: [GSAP](https://greensock.com/gsap/)
- **Iconos**: Heroicons & Custom SVGs
- **Fuentes**: Inter & Modern Typography

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/BeaCao/helio-vision-ecommerce.git
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```
